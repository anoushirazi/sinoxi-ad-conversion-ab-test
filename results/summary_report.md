# Sinoxi Ad Conversion A/B Test - Summary Report

## Project Objective
To determine if the new ad delivery strategy improves user conversion rates compared to the current method.

## Methodology
- Collected user conversion data divided into control and treatment groups.
- Conducted two-proportion Z-test to evaluate conversion differences.
- Calculated 95% confidence intervals for conversion rates.

## Key Results
| Metric                    | Value        |
|--------------------------|--------------|
| Control group conversion rate | 2.71%       |
| Treatment group conversion rate | 1.79%       |
| Conversion lift           | -34.14%      |
| Statistical significance (p-value) | 0.0004      |

## Business Recommendation
The treatment group demonstrates a statistically significant decrease in conversion rate. It is recommended to retain the current ad delivery strategy and investigate the causes for the performance drop before adopting the new method.
